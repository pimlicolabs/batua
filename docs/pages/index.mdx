## Key Features

- **Easy Integration**: Embedded smart account secured by passkeys
- **Transaction Sponsoring**: Support for sponsoring transactions  
- **Batch Transactions**: Support for batching multiple transactions
- **Complete Ownership**: You have the complete ownership of the code
- **Theme Integration**: Embeds into your application's theme due to shadcn
- **Wide Compatibility**: Works with wagmi, viem, ethers, privy, dynamic, and more
